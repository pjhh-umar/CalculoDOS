Ejercicios propuestos primer examen parcial

Ejercicio resuelto

Demuestra que toda sucesión constante es convergente.

Ejercicio

Encuentra una fórmula para el $n$-ésimo término de cada una de las siguientes sucesiones:

  1. $\displaystyle{-2, 4, -6,8, \ldots }$

  2. $\displaystyle{\frac{1}{1}, \frac{-1}{8}, \frac{1}{27}, \frac{-1}{64}, \ldots }$

  3. $\displaystyle{\frac{2}{6}, \frac{3}{7}, \frac{4}{8}, \ldots }$

  4. $\displaystyle{\frac{1}{4}, \frac{2}{5}, \frac{1}{2},\frac{4}{7}, \frac{5}{8}, \frac{2}{3}, \frac{7}{10} \ldots }$

  5. $\displaystyle{1,1,\frac{3}{4}, \frac{1}{2}, \frac{5}{16}, \frac{3}{16},\frac{7}{64}, \frac{1}{16}, \frac{9}{256} \ldots }$

  6. $\displaystyle{\frac{2}{1}, \left(\frac{3}{2}\right)^2,\left(\frac{4}{3}\right)^3, \left(\frac{5}{4}\right)^4, \ldots }$

Ejercicio

Determine cuales de las siguientes sucesiones convergen o cero. Justifica clara y detallamente tus respuestas

  1. $\displaystyle{\left( 2^{n} \right)_{n=1}^{\infty}}$

  2. $\displaystyle{\left( \frac{-1}{2^{n}} \right)_{n=1}^{\infty}}$

  3. $\displaystyle{\left( \frac{n^{2}}{n^{2} + 1} \right)_{n=1}^{\infty}}$

Ejercicio

Considera la sucesión $\left(a_{n} \right)_{n \geq 1}$ tal que $a_{n} := \displaystyle{ \frac{n}{n+1} }$, para todo $n \in \mathbb{N}$.

  • Utiliza la definición de convergencia de sucesiones en $\mathbb{R}$ para probar que $\displaystyle{\lim_{n\to \infty } a_{n}} =1$.

  • Luego, encuentra $N \in \mathbb{N}$ tal que:

    1. $|a_{n} - 1| < 0.0001$ para todo $n\geq N$

    2. $|a_{n} - 1| < 0.00001$ para todo $n\geq N$

Ejercicio

Considera la sucesión $\left(b_{n} \right)_{n \geq 1}$ tal que $b_{n} := \displaystyle{ \left(\frac{1}{3}\right)^{n} }$, para todo $n \in \mathbb{N}$.

  • Utiliza la definición de convergencia de sucesiones en $\mathbb{R}$ para probar que $\displaystyle{\lim_{n\to \infty } b_{n}} =0$

  • Luego, encuentra $N \in \mathbb{N}$ tal que:

    1. $|b_{n}| < 10^{-2}$ para todo $n\geq N$

    2. $|b_{n}| < 10^{-5}$ para todo $n\geq N$

Ejercicio

Utilizando la definicion de convergencia de sucesiones determine si las siguientes sucesiones convergen o divergen, en caso de que sean convergentes establece el límite $L$ y prueba detalladamente que en efecto $\displaystyle{\lim_{n\to \infty}a_{n}}=L$. Justifica claramente tus respuestas

  1. $\displaystyle{\left( 12 \right)_{n=1}^{\infty}}$

  2. $\displaystyle{\left( n+5 \right)_{n=1}^{\infty}}$

  3. $\displaystyle{\left( \frac{1}{n+2} \right)_{n=1}^{\infty}}$

  4. $\displaystyle{\left( \frac{5n-1}{12n+9} \right)_{n=1}^{\infty}}$

  5. $\displaystyle{\left( -2^{-n} \right)_{n=1}^{\infty}}$

  6. $\displaystyle{\left( n^{-2} \right)_{n=1}^{\infty}}$

  7. $\displaystyle{\left( (-1)^{n} n^{2} \right)_{n=1}^{\infty}}$

  8. $\displaystyle{\left( 10^{-\frac{1}{n}} \right)_{n=1}^{\infty}}$

  9. $\displaystyle{\left( \frac{3n}{n+ 2} \right)_{n=1}^{\infty}}$

  10. $\displaystyle{\left( \frac{n}{n+ n^{-1}} \right)_{n=1}^{\infty}}$

  11. $\displaystyle{\left( \frac{n}{n^{2} + 1} \right)_{n=1}^{\infty}}$

  12. $\displaystyle{\left( \frac{2n}{n^{2} + 1} \right)_{n=1}^{\infty}}$

  13. $\displaystyle{\left( \frac{2n^{2}}{n^{2} + 1} \right)_{n=1}^{\infty}}$

  14. $\displaystyle{\left( \frac{n^{2}}{n + 1} \right)_{n=1}^{\infty}}$

  15. $\displaystyle{\left( \frac{(-1)^{n}n}{n + 1} \right)_{n=1}^{\infty}}$

  16. $\displaystyle{\left( \frac{3n+5}{7n-4} \right)_{n=1}^{\infty}}$

Ejercicio

Sea $a\in \mathbb{R}$ tal que: $|a|< 1$. Demuestra que la sucesión $\left( a^{n}\right)_{n \geq 1}$ es convergente y converge a $0$

Hint: Puedes considera los casos: $a=0$ y $a\not= 0$ o bien los casos: $a=0$, $-1< a < 0$ y $0 < a < 1$

Ejercicio

Sea $f$ una función definida en $\mathbb{R}_{+}$. Demuestra que si $\displaystyle{\lim_{x \to \infty}f(x) }$ existe, entonces la sucesión $\left( a_{n}\right)_{n\geq 1}$ tal que $a_{n}:= f(n)$ para todo $n\in \mathbb{N}$, es convergente y converge al mismo límite, es decir, si existe $L\in \mathbb{R}$ tal que $\displaystyle{\lim_{x\to \infty} f(x)}$, entonces $$\lim_{n\to \infty} a_{n}= L$$

Ejercicio

Sea $\left( a_{n}\right)_{n\geq 1}$ es una sucesión de números enteros i.e. $a_{n} \in \mathbb{Z}$ para todo $n\in \mathbb{N}$. Si $(a_{n})_{n\in \mathbb{N}}$ es convergente con $\displaystyle{\lim_{n\to \infty}} a_{n}=L$, utiliza la definición de sucesión convergente para probar que existe $N\in \mathbb{N}$ tal que $a_{n}=L$ para todo $n \geq N$.

Ejercicio

Da un ejemplo de dos sucesiones de números reales $(a_n)_{n=1}^\infty$ y $(b_n)_{n=1}^\infty$ convergentes, tales que $a_n < b_n$ para toda $n\in \mathbb{N}$ y sin embargo $$\lim_{n\to \infty} a_n = \lim_{n\to \infty} b_n.$$

Ejercicio

Sea $(a_n)_{n=1}^\infty$ una sucesión en $\mathbb{R}$. Prueba que si $(a_n)_{n=1}^\infty$ es monótona decreciente y acotada inferiormente, entonces la sucesión $(a_n)_{n=1}^\infty$ es convergente.

Hint: La sucesion converge a $\beta:=\inf\{ a_{n} \colon n \in \mathbb{N} \}$. Justifica adecuadamente porque existe $\beta$

Ejercicio

Sea $(a_n)_{n=1}^\infty$ una sucesión de números reales convergente con $\displaystyle{\lim_{n\to \infty} a_{n}} = L$. Prueba que:

  1. Si $L>0$, entonces existe $N\in \mathbb{N}$ tal que $a_{n} >0$ para toda $n\geq N$

  2. Si $L < 0$, entonces existe $N\in \mathbb{N}$ tal que $a_{n} < 0$ para toda $n\geq N$

  3. Si $L\not= 0$, entonces existe $N\in \mathbb{N}$ tal que $a_{n} \not= 0$ para toda $n\geq N$

Ejercicio

Sea $(a_n)_{n=1}^\infty$ una sucesión de números reales. Prueba que $\left( a_{n}\right)_{n\geq 1}$ converge a $0$ si y sólo si la sucesión $\left(|a_{n}|\right)_{n\geq 1}$ converge a $0$.

Ejercicio

  1. Sea $(a_n)_{n=1}^\infty$ una sucesión de números reales. Prueba que $\left( a_{n}\right)_{n\geq 1}$ converge a $a$ entonces la sucesión $\left(|a_{n}|\right)_{n\geq 1}$ converge a $|a|$.

  2. ¿Es verdad que si la sucesión $\left(|a_{n}|\right)_{n\geq 1}$ converge, entonces la sucesión $\left(a_{n}\right)_{n\geq 1}$ también converge? Prueba o da un contraejemplo.

Ejercicio

Demuestra que la sucesion $\displaystyle{ \left(\frac{(−1)^n}{n}\right)_{n=1}^{\infty}}$ converge a cero.

Ejercicio

Sea $c\in \mathbb{R}$ tal que $|c|>1$. Demuestra que la sucesion $(c^{n})_{n\in \mathbb{N}}$ no es una sucesión convergente.

Ejercicio

Sean $(a_n)_{n=1}^\infty$ y $(b_n)_{n=1}^\infty$ sucesiones de números reales tales que $(a_n)_{n=1}^\infty$ es convergente con $\displaystyle{\lim_{n\to \infty}} a_{n}=0$ y $(b_n)_{n=1}^\infty$ es acotada. Prueba que la sucesión $(a_n\cdot b_n)_{n=1}^{\infty}$ converge a $0$

Ejercicio

Sea $A \subset \mathbb{R}$ tal que $A\not= \emptyset$ y $A$ acotado superiormente. Prueba que si $\alpha = \sup(A)$, entonces existe una sucesión $(a_{n})_{n=1}^{\infty}$ de números en $A$ i.e. $a_{n} \in A$ para todo $n\in \mathbb{N}$ tal que $(a_{n})_{n=1}^{\infty}$ converge a $\alpha$

Ejercicio

Sean \(a,b,c \in \mathbb{R}\) tales que \(a< b< c\). Demuestra que para todo \(L\in \mathbb{R}\) se satisface: \[ |b-L| \leq |a-L|+|c-L|. \]

Sugerencia: Considera los casos \(L\leq a\), \(a< L < c \) y \(c \leq L\) junto con la desigualdad del triángulo.

Ejercicio

  1. Demuestra que $$ L=\lim_{n\to \infty} a_{n+1}.$$
  2. Más en general, toma \(p\in \mathbb{N}\) fijo y define una nueva sucesión \(b_n:=a_{n+p}\). Demuestra que la sucesión \((b_n)_{n=1}^\infty\) converge si y sólo si \((a_n)_{n=1}^\infty\) converge y que ambas tienen el mismo límite.

Ejercicio

Desigualdad de Bernoulli

Utiliza inducción matemática para probar que si $x>-1$, entonces para todo $n\in \mathbb{N}$ se satisface: $$(1+nx)\leq (1+x)^{n}.$$

Ejercicio

Para cada uno de los incisos, justifica detalladamente lo establecido.

  1. $\displaystyle{\lim_{n \to\infty} \left(\frac{1}{2^n}\right)}=0$

  2. $\displaystyle{\lim_{n \to\infty} \left(\frac{1}{2n-1}\right)}=0$

  3. $\displaystyle{\lim_{n \to\infty} \left(\frac{(-1)^{n+1}}{n}\right)}=0$

  4. $\displaystyle{\lim_{n \to\infty} \left(\frac{(-1)^{n+1}}{n}\right)}=0$

  5. $\displaystyle{\lim_{n \to\infty} \left(1 - \frac{1}{10^n}\right)=1}$

  6. $\displaystyle{\lim_{n \to\infty} \left(\frac{2^n}{n!}\right)=0}$

  7. $\displaystyle{\lim_{n \to\infty} \left(\frac{n!}{n^n}\right)=0}$

  8. $\displaystyle{\lim_{n \to\infty} \left(\frac{2n^5 - 3n + 20}{5^6+2}\right)=0}$

Ejercicio

Sean $(a_n)_{n=1}^\infty$ una sucesión de números reales tales que $(a_n)_{n=1}^\infty$ es convergente con $\displaystyle{\lim_{n\to \infty}} a_{n}=L$ Para cada $n\in \mathbb{N}$ definamos $s_{n}:= \displaystyle{\frac{a_{1}+a_{2} + \cdots + a_{n}}{n}}$. Demuestra que la sucesión $\left(s_n\right)_{n=1}^{\infty}$ converge a $L$, es decir, prueba que se satisface: $\displaystyle{\lim_{n\to \infty}\displaystyle{\frac{a_{1}+a_{2} + \cdots + a_{n}}{n}} =L}$

Ejercicio

Para cada uno de los siguientes incisos, determina si la sucesión dada es monotona creciente, monotona decreciente o ninguna de estas. Justifica detalladamente tu respuesta.

  1. $\displaystyle{\left(\frac{2n}{3n+1}\right)_{n=1}^{\infty}}$

  2. $\displaystyle{\left(\frac{5n-2}{4n+1}\right)_{n=1}^{\infty}}$

  3. $\displaystyle{\left(\frac{3n}{1+3^n}\right)_{n=1}^{\infty}}$

  4. $\displaystyle{\left(\frac{n}{2^n}\right)_{n=1}^{\infty}}$

  5. $\displaystyle{\left(\frac{n!}{2^n}\right)_{n=1}^{\infty}}$

  6. $\displaystyle{\left(\frac{n^2-1}{n}\right)_{n=1}^{\infty}}$

  7. $\displaystyle{\left(\frac{n^n}{n!}\right)_{n=1}^{\infty}}$

  8. $\displaystyle{\left(\prod_{k=1}^{n}\frac{2k-1}{2k}\right)_{n=1}^{\infty}}$

Ejercicio

Sea $(m_{n})_{n=1}^{\infty}$ una sucesión de números naturales i.e. $m_{n}\in \mathbb{N}$ para todo $n\in \mathbb{N}$. Prueba por inducción que $m_{n}\geq n$ para todo $n\in \mathbb{N}$.
