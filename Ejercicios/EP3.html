Ejercicio

Calcular las siguientes integrales.

  1. $\displaystyle{\int_{-2}^{1} 4 dx}$.
  2. $\displaystyle{\int_{-\pi}^{2\pi} \pi dx}$.
  3. $\displaystyle{\int_{-3}^{3} 2x dx}$.
  4. $\displaystyle{\int_{-1}^{0} x^{99} dx}$.
  5. $\displaystyle{\int_{-2}^{1}(3x+4)dx}$.
  6. $\displaystyle{\int_{6}^{8}(7x-8)dx}$.
  7. $\displaystyle{\int_{-2}^{2}(2 - |x|)dx}$.
  8. $\displaystyle{\int_{-2}^{3}(2x+4)dx}$.
  9. $\displaystyle{\int_{-3}^{2}(4x+7)dx}$.
  10. $\displaystyle{\int_{-1}^{2}|x|dx}$.
  11. $\displaystyle{\int_{-2}^{1}|x|dx}$.
  12. $\displaystyle{\int_{-a}^{a}|x|dx}, a >0$.
  13. $\displaystyle{\int_{-1}^{1}(2-|x|)dx}$.
  14. $\displaystyle{\int_{-2}^{5} (3 - x -2|x|)dx}$.
  15. $\displaystyle{\int_{0}^{6} (|12 - 4x| -4)dx}$.
  16. $\displaystyle{\int_{-2}^{2}(x^2 -1)(x^2-4)dx}$.
  17. $\displaystyle{\int_{-3}^{3}(9x -4x^2)dx}$.
  18. $\displaystyle{\int_{a}^{a^2}x^2dx}$.
  19. $\displaystyle{\int_{-a}^{1}(x^2+x)dx}$.
  20. $\displaystyle{\int_{-1}^{2}|x^2|dx}$.

Ejercicio

Sean $f,g \colon \mathbb{R} \to \mathbb{R}$ funciones continuas tales que para todo $a \in \mathbb{R}$ se satisface $$\int_{-a}^{a} f(x) dx = \int_{-a}^{a} g(x) dx$$ Da un argumento intiutivo para mostrar que $f(0) = g(0)$. Explica tu idea mediante una gráfica.

Ejercicio

Para cada uno de los siguientes ejercicios menciona claramente

  1. Si $\displaystyle{\int_{3}^{5} f(x) dx = 2}$, encuentra $\displaystyle{\int_{5}^{3} -4f(x)}$
  2. Si $\displaystyle{\int_{5}^{-3} \left(\frac{-3f(x)}{4}\right) dx = 7 }$, encuentra $\displaystyle{\int_{5}^{-3} (6f(x) + 1) dx}$
  3. Si $\displaystyle{\int_{1}^{4} \left(\frac{f(x)}{3}\right) dx = 2 }$, encuentra $\displaystyle{\int_{1}^{4} 3f(x) dx}$
  4. Si $\displaystyle{\int_{2}^{1} f(x) dx = 0}$ y $\displaystyle{\int_{1}^{2} g(x) dx= 0}$, encuentra $\displaystyle{\int_{2}^{1} (f(x) - 3g(x) + 5)} dx$
  5. Si $\displaystyle{\int_{6}^{8} (3f(x) - x)dx = 6}$ y $\displaystyle{\int_{8}^{6} (2x + 4g(x)) dx = -8}$, encuentra $\displaystyle{\int_{8}^{6} (f(x) - 5g(x))} dx$
